🎯 Number Guessing Game

An interactive and fun browser-based Number Guessing Game built using HTML, CSS, and JavaScript.

🚀 Features

🎲 Random number generation

📈 Multiple difficulty levels (Easy / Hard)

📢 "Too High" / "Too Low" hints

🧮 Dynamic score tracking

🎉 Win & Game Over alerts

💻 Simple and responsive UI

🕹️ How to Play

Choose a difficulty level.

Enter a number within the given range.

Click the Guess button.

Get instant feedback:

📉 Too High

📈 Too Low

🎉 Correct Guess

Score decreases for every wrong attempt.

Guess correctly before score reaches zero!

🛠️ Technologies Used

HTML

CSS

JavaScript (DOM Manipulation & Logic)
